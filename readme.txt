Interactive map of New York WordPress plugin with counties.

== Description ==

Interactive map of New York WordPress plugin with clickable counties. Edit colors, link, etc for each county through an easy to use dashboard with a Visual Editor.

Features:

*	Does NOT require ANY coding knowledge.
*	EASY to use dashboard.
*	Bulk Edit option.
*	Works on desktop and mobile devices.

You may be interested in the [Premium Interactive Map of New York](https://www.wpmapplugins.com/states/interactive-map-of-new-york-wordpress-plugin.html) which has more options.

Other interesting maps: [World](https://www.wpmapplugins.com/interactive-world-map-by-countries-wordpress-plugin.html), [US](https://www.wpmapplugins.com/interactive-us-map-wordpress-plugin.html), [Canada](https://www.wpmapplugins.com/interactive-map-of-canada-wordpress-plugin.html), [Other States](https://www.wpmapplugins.com/states/individual-us-state-maps-package-wordpress-plugins.html), [All Maps](https://www.wpmapplugins.com/all-maps.html).

== Installation ==

1. Install and activate the plugin .zip file through your WordPress 'Plugins' screen directly.
2. Now, you will see a new menu button in the left side bar called 'Europe Map ORG', click it and customize your map then save the changes.
4. Use this shortcode `[nyus_map_org]` to display the map in any page or post.

== Frequently Asked Questions ==

= Can I set a county as inactive? =

Yes, simply uncheck the (active) checkbox beside that county to make it inactive. Also you can give it some grey color.

= Can I use the shortcode in different pages? =

Yes, you can use the shortcode in different pages to display the same map.

= Can I create another map with different colors and links? =

No, but the [Premium Interactive Map of New York](https://www.wpmapplugins.com/states/interactive-map-of-new-york-wordpress-plugin.html) comes with 3 installable instances of the plugin.

= Does the map have a hover popup? =

No, but the [Premium Interactive Map of New York](https://www.wpmapplugins.com/states/interactive-map-of-new-york-wordpress-plugin.html) has that option. It also has an option to open a modal window in case you want to add more information about the counties.
